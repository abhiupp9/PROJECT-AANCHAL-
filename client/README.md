# AANCHAL Client — React + Vite Frontend

This is the frontend for [Project AANCHAL](https://github.com/abhiupp9/PROJECT-AANCHAL-), built with React 19 and Vite.

## Development

```bash
npm install
npm run dev
```

> Requires the backend (`server/`) to be running on port 5000.

See the root [README.md](../README.md) for full project documentation.
